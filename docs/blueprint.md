# **App Name**: Misión Facu 9

## Core Features:

- Carga del Mundo 3D: Una pantalla de carga a pantalla completa con un logo tridimensional animado 'FACU 9', monedas giratorias, y una barra de progreso. Incluye un botón gigante '¡START!' para iniciar el juego.
- Mapa de la Aldea Interactivo: Exploración de un 'Mapa de la Aldea' para mostrar los detalles del evento, utilizando un diseño de tarjetas estilo 'Cajas de Diálogo de Videojuego'.
- Diálogo con los Personajes: Muestra la Fecha, Hora y Ubicación del cumpleaños al hacer clic en íconos flotantes estilo 3D (un reloj, un mapa, un calendario).
- Búsqueda de las Llaves Mágicas: Una 'Mini-Misión' que requiere recolectar 3 llaves amarillas visualmente distribuidas en la página para desbloquear el formulario de confirmación, con seguimiento de estado mediante React.
- Portal de Confirmación de Misión: Animación para revelar el formulario de 'Confirmar Misión' (RSVP) una vez que todas las llaves han sido recolectadas.
- Registro de Aventurero: Formulario de 'Confirmar Misión' para el 'Nombre del Jugador' y un checkbox de confirmación, con manejo básico del envío de datos.
- Nivel Completado y Recompensa: Una pantalla de '¡Nivel Completado!' al enviar el RSVP, con un botón dorado de 'recompensas' simulando una victoria en el juego.

## Style Guidelines:

- Esquema de color general: Luminoso, que evoca una aventura alegre de un videojuego low-poly.
- Fondo base principal: Sky Blue (#87CEEB), para un ambiente de cielo abierto.
- Textos y bordes principales: Teddy Brown (#8B4513), aportando una sensación de tierra y estabilidad.
- Botones de acción y elementos interactivos: Grass Green (#7CFC00), para destacar acciones importantes y energéticas.
- Elementos de recompensa y finalización: Golden Coin (#FFD700), simbolizando tesoros y logros.
- Fondos de tarjetas y paneles de información: Cloud White (#FFFFFF), proporcionando un lienzo limpio para los detalles.
- Fuentes para títulos y encabezados: 'Milky Nice', para un estilo lúdico y destacado. Note: currently only Google Fonts are supported.
- Fuentes para párrafos y texto del cuerpo: 'Quicksand' (sans-serif), para legibilidad y un toque moderno. Note: currently only Google Fonts are supported.
- Íconos estilo 3D y low-poly, específicamente para reloj, mapa y calendario, que floten y encajen con la temática del videojuego.
- Diseño inspirado en un videojuego de plataformas 3D Low-Poly, utilizando sombras gruesas (box-shadow) en contenedores y botones para simular profundidad.
- Disposición de contenido que imita un 'Mapa de la Aldea' y 'Cajas de Diálogo de Videojuego' para los detalles del evento.
- Animaciones de carga para el logo, monedas giratorias y barra de progreso al inicio.
- Transiciones fluidas, incluyendo un deslizamiento de pantalla al iniciar y una animación de revelación para el formulario RSVP.
