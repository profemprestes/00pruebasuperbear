# Contexto del Proyecto: Invitación Gamificada "Nivel Facu"

## 1. Perfil

- **Nombre:** Facundo ("Facu").
- **Personalidad:** Deportista (fanático del básquet), creativo, aventurero.
- **Narrativa:** El cumpleañero es el "Bear" protagonista del juego. Los invitados son aliados que él necesita para una nueva misión.
- **Tono:** Divertido, creativo, épico y aventurero (estilo _Quest_).

## 2. Público

- **Perfil Mixto:** Niños (amigos de Facu) y Adultos (padres/tutores).
- **UX Diferenciada:**
  - **Adultos:** Flujo técnico, informativo, directo al grano (logística y confirmación).
  - **Niños:** Flujo gamificado, lúdico, con mini-juegos y retos interactivos.
- **Objetivo emocional:** Sorpresa, curiosidad y diversión inmediata desde el primer contacto.

## 3. Estructura de Pantallas (Flujo de Nivel)

1.  **Pantalla de Carga (Intro):** Gamificada. Divide el camino entre "Modo Guardián" (Adultos) y "Modo Aventurero" (Niños).
2.  **Pantalla de Bienvenida (Hub):**
    - _Adultos:_ Info clara y directa.
    - _Niños:_ Bienvenida inmersiva con mini-juegos centrados en Facu y el universo _Super Bear_.
3.  **Pantalla de "Misión" (Detalles):** Fecha, lugar y mapa.
4.  **Pantalla de "Equipamiento" (RSVP/Regalos):** Formulario de confirmación. Guía creativa de gustos del cumpleañero (wishlist oculta) para orientar regalos.
5.  **Pantalla Final (Recompensa):** Cierre temático y datos sorprendentes sobre Facu.

## 4. Estilo Visual y Técnico

- **Colores (Paleta _Sky World_):**
  - `Primary` (#0C6780 / #87CEEB): Contenedores principales.
  - `Secondary` (#705D00 / #FCD400): Elementos interactivos y CTAs.
  - `Tertiary` (#934B19): Detalles de terreno.
- **Tech Stack:** Next.js 16 (App Router), TypeScript, Tailwind CSS.
- **Interactividad:** Animaciones avanzadas, efectos de sonido al interactuar, efecto de cámara (tipo giroscopio).
- **Librería sugerida:** _Framer Motion_ (para transiciones fluidas de pantalla a pantalla).

## 5. Idea Maestra

- **Gamificación total:**
  - Mini-juegos ocultos.
  - Cuenta regresiva (tipo "temporizador de bomba" o "reloj de partido").
  - Desbloqueo de información (el usuario debe completar un reto, como encestar balones digitales, para ver el mapa o la ubicación).
