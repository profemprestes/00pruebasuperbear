# Instrucciones de Creación
Se me ha solicitado actuar como un Experto Frontend y UX/UI para generar una o varias versiones en HTML del proyecto "Gran Facu Aventura Voxel", basándome en los archivos de referencia y `BLUEPRINT_nueva_web.md`.

Crearemos un archivo `index_v2_unificado.html` que contendrá toda la experiencia, combinando los distintos archivos provistos en una sola SPA (Single Page Application) sin recargar la página.

El flujo a implementar será:
1. Pantalla de Bienvenida (Estilo App/Game loading)
2. Diálogo Introductorio (Facundo / Shicka) para elegir Modo.
3. Bifurcación:
   - Modo Adulto (Directo a información + RSVP)
   - Modo Pro (Arcade World con mapa, y minijuegos, terminando en el RSVP/Inventario de regalos)

Usaremos CSS vainilla y JavaScript nativo. Se implementarán las tipografías Fredoka One y Nunito, así como la paleta de colores indicada.
